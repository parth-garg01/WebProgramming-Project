1. Introduction
The Portfolio Website is a simple personal web project created to present my academic background, technical skills, projects, research work, and contact details in one place. The website acts as an online profile where anyone can quickly learn about me, my education at VIT Vellore, my internship experience, and the work I have done in different technical areas. 

The website is built using HTML, CSS, and JavaScript. It is designed to be clean, easy to read, and responsive so that it works properly on both laptop and mobile screens. The main aim of this project is to create a professional but student-friendly portfolio that is not too flashy and not too basic.

This project demonstrates my understanding of frontend web development fundamentals, including page structure, styling, responsive design, and simple JavaScript-based interactivity.
